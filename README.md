# Task 03 – Prompting for Task Automation

## Objective
To design a prompt that automates a real-world task such as summarizing text and extracting structured information.

---

## Task Chosen
Text Summarization and Key Point Extraction

---

## Base Prompt
"Summarize the given text and extract key points in bullet format."

---

## Input-Output Examples

### Example 1
**Input:**  
Artificial Intelligence is transforming industries by automating tasks, improving efficiency, and enabling data-driven decisions.

**Output:**  
Summary:  
AI is changing industries by automating work and improving efficiency.

Key Points:
- Automates tasks  
- Improves efficiency  
- Enables data-driven decisions

---

### Example 2
**Input:**  
Online education allows students to learn anytime and anywhere using digital platforms.

**Output:**  
Summary:  
Online education provides flexible learning through digital platforms.

Key Points:
- Learn anytime  
- Learn anywhere  
- Uses digital platforms

---

### Example 3
**Input:**  
Good communication skills help build relationships, improve teamwork, and increase career success.

**Output:**  
Summary:  
Communication skills are important for relationships, teamwork, and career growth.

Key Points:
- Builds relationships  
- Improves teamwork  
- Supports career success

---

## Reflection
Using a clear and consistent prompt produced reliable outputs across different inputs. Minor improvements were made by specifying the output format clearly.

---
